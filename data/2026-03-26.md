### [1] [Leveraging Computerized Adaptive Testing for Cost-effective Evaluation of Large Language Models in Medical Benchmarking](https://arxiv.org/abs/2603.23506)
*Tianpeng Zheng,Zhehan Jiang,Jiayi Liu,Shicong Feng*

Main category: cs.CL

TL;DR: This paper introduces and validates a computerized adaptive testing (CAT) framework, based on item response theory (IRT), to rapidly and reliably evaluate large language models’ medical knowledge using only a tiny fraction of questions while preserving accuracy and rank ordering.


<details>
  <summary>Details</summary>
Motivation: Large language models are increasingly used in healthcare, but current evaluation methods rely on static, fixed test sets that are expensive to reuse, susceptible to data contamination (models seeing test items during training), and lack rigorous psychometric properties for precise, longitudinal performance tracking. There is a need for a scalable, low-cost, and statistically calibrated way to frequently benchmark LLMs’ foundational medical knowledge without administering long test batteries each time.

Method: The authors design an item response theory–based computerized adaptive testing framework using a human-calibrated medical question bank. In phase one, they run Monte Carlo simulations to identify optimal CAT configurations (e.g., item selection rules, stopping criteria, target standard errors). In phase two, they empirically test 38 LLMs: each model answers the full item bank and then an adaptive test in which items are dynamically selected based on the model’s evolving ability estimate, with testing stopping once the standard error of the ability estimate drops to 0.3 or below. They compare proficiency estimates, item usage, evaluation time, and inter-model rankings between the full-bank and CAT conditions.

Result: CAT-based proficiency estimates correlate almost perfectly with full-bank estimates (r = 0.988), despite using only about 1.3% of the total items. This adaptive approach reduces evaluation time from hours to minutes per model and substantially cuts token usage and computational cost, yet still preserves the relative performance ranking among different LLMs.

Conclusion: An IRT-grounded CAT framework can provide fast, low-cost, and psychometrically robust benchmarking of LLMs’ standardized medical knowledge. The method is suitable for standardized pre-screening and continuous monitoring of models but is not intended to replace real-world clinical validation or prospective safety studies, which remain necessary before clinical deployment.

Abstract: The rapid proliferation of large language models (LLMs) in healthcare creates an urgent need for scalable and psychometrically sound evaluation methods. Results show that CAT-derived proficiency estimates achieved a near-perfect correlation with full-bank estimates (r = 0.988) while using only 1.3 percent of the items. Abstract: While Masked Diffusion Language Models (MDLMs) relying on token masking and unmasking have shown promise in language modeling, their computational efficiency and generation flexibility remain constrained by the masking paradigm.
