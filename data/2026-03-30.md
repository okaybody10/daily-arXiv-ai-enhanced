### [2] [Density-aware Soft Context Compression with Semi-Dynamic Compression Ratio](https://arxiv.org/abs/2603.25926)
*Yijiong Yu,Shuai Yuan,Jie Zheng,Huazheng Wang,Ji Pei*

Main category: cs.CL

TL;DR: They propose a semi-dynamic, density-aware context compression method for LLMs that chooses from a small set of discrete compression ratios, improving efficiency–accuracy trade-offs over static compression.


<details>
  <summary>Details</summary>
Motivation: Processing very long contexts in LLMs is computationally expensive. Soft context compression can reduce cost by encoding long sequences into fewer latent tokens, but current methods use a single, uniform compression ratio for all inputs. Natural language has highly variable information density across sequences and segments, so uniform compression is suboptimal. A dynamic, density-aware ratio is appealing, but in practice LLMs struggle when required to handle operations governed by continuous, input-dependent structural hyperparameters (like arbitrary per-input ratios). This motivates a new framework that can adapt compression to information density while remaining easy to train and robust for LLMs.

Method: They introduce Semi-Dynamic Context Compression. The key component is a Discrete Ratio Selector that predicts a suitable compression target based on the intrinsic information density of the input and then quantizes this prediction to one of a predefined set of discrete compression ratios. Instead of using a fully continuous ratio, they restrict the model to a small, discrete menu. The selector is trained jointly with the compressor, using synthetic data where available summaries and their lengths serve as proxy labels to infer the desired compression ratio. The actual compression backbone uses mean pooling to map long contexts to a smaller set of latent tokens. The whole system is trained end-to-end so that the selector learns which ratio to pick for different density patterns while the compressor learns to preserve task-relevant information under those ratios.

Result: Across extensive evaluations, their density-aware semi-dynamic framework consistently beats static, uniform-ratio baselines on the trade-off between performance and compression (i.e., for a given computational budget it yields higher task accuracy, or for a given accuracy it allows stronger compression). Even with a simple mean pooling compressor, the method establishes a strong Pareto frontier for context compression, indicating state-of-the-art or near state-of-the-art efficiency–accuracy balance among competing techniques.

Conclusion: Semi-Dynamic Context Compression shows that discretizing the space of compression ratios and learning a density-aware selector is an effective way to overcome the training difficulties of fully dynamic, continuous-parameter compression in LLMs. By adapting compression levels to intrinsic information density while remaining structurally simple (mean pooling backbone, discrete ratios), the framework provides robust, scalable context compression that outperforms static methods and sets a strong benchmark for future compression approaches. The released code, data, and weights support further research and practical adoption.

Abstract: Soft context compression reduces the computational workload of processing long contexts in LLMs by encoding long context into a smaller number of latent tokens. However, existing frameworks apply uniform compression ratios, failing to account for the extreme variance in natural language information density. While adopting a density-aware dynamic compression ratio seems intuitive, empirical investigations reveal that models struggle intrinsically with operations parameterized by input dependent, continuous structural hyperparameters. To resolve this pitfall, we introduce Semi-Dynamic Context Compression framework. Our approach features a Discrete Ratio Selector, which predicts a compression target based on intrinsic information density and quantizes it to a predefined set of discrete compression ratios. It is efficiently jointly trained with the compressor on synthetic data, with the summary lengths as a proxy to create labels for compression ratio prediction. Extensive evaluations confirm that our density-aware framework, utilizing mean pooling as the backbone, consistently outperforms static baselines, establishing a robust Pareto frontier for context compression techniques. Our code, data and model weights are available at https://github.com/yuyijiong/semi-dynamic-context-compress
